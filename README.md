# Motorcycle School Instructor App

Мобільний додаток для інструкторів мотошколи для відслідкування прогресу учнів під час навчання.

## Можливості

- 📋 Управління списком учнів
- 🎯 Відмічання виконаних вправ по кожному уроку
- 📊 Прогрес-бар для кожного учня
- 📱 Мобільний інтерфейс для iOS та Android
- 💾 Локальне зберігання даних

## Технології

- **React Native** з Expo
- **SQLite** для локального сховища
- **TypeScript** для типізації

## Встановлення

```bash
npm install
npm start
```

## Структура проекту

```
src/
├── components/       # Компоненти UI
├── screens/         # Екрани додатка
├── services/        # Бізнес-логіка та БД
├── types/           # TypeScript типи
└── navigation/      # Навігація
```
